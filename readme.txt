1、程序的主函数是judge.java，点击运行之后会显示更改的对话框
2、运行环境：
	Eclipse Java 2019-03
	jdk12_也就是课上电脑的环境
3、目录介绍：
	文件夹中包含了四个类
	judge.java是主程序
	frame.java包含了界面制作的相关属性
	distance.java是计算机距离以及进行排名选出top-n的类
	MyListModel.java也是制作面板相关的一个小类，主要用在面板中滚动选择窗口。
4、程序运行中会出现这几种情况：
	4.1、没有错误：没有窗口弹出，程序会输出“恭喜恭喜啦~~当前没有任何错误的单词，修改完毕！”。
	4.2、单词错误且出现在top-n选项中，选择单词――>写入词典――>确定。这个单词就被成功修改，并且词典也增加了一个。
	4.3、单词错误但没有出现在top-n中，在右边写单词――>写入词典――>确定。这个单词就被成功修改，并且词典也增加了一个。
	4.4、单词正确但没有出现在词典中，在右边写该单词――>写入词典――>确定。这个单词就成功加入词典。
5、执行完毕之后，原文得到了修改，并且词典中加入了新的单词。每执行一步都可以看到效果。
5、注意不能同时既选择了单词又输入单词。只能选择一种。
